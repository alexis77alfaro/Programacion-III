Introduccion.

El presente trabajo esta enfocado en explicar el proceso de instalación y el uso de los diferentes CMS(Sistemas de gestión de contenidos) utilizando un servisor local, se presentaran manuales de instalación, videos tutoriales con el fin de tener una mayor comprensión en torno a la utilización de los CMS.p

Sabiendo que los CMS son software desarrollados para que cualquier usuario pueda administrar y gestionar contenidos de una web con facilidad y sin conocimientos de programación Web, queremos compartir esta información para que este a dispocición de toda aquella persona interesada en crear un sitio web facíl y sencillo de administrar y utilizar.

    ¿Qué son los CMS?

    Un CMS es un software desarrollado para que cualquier usuario pueda administrar y gestionar contenidos de una web con facilidad y sin conocimientos de programación Web. Previamente un programador deberá desarrollar la página web en base al CMS más apropiado para el tipo de web, dejando las tareas posteriores de gestión en manos del usuario final.
    No todas las páginas Web son iguales, y para ello tenemos varios tipos de CMS según el tipo de página que necesitemos. Los hay para blogs, páginas corporativas, inmobiliarias, tiendas on-line, noticias o magazines, contenidos multimedia, y así un largo etc.

    ¿Qué ventajas proporciona el uso de CMS?

    • Los programadores y la comunidad están organizados para desarrollar y lanzar nuevas versiones de estos CMS de forma periódica.
    • Están diseñados para que una vez implementado el proyecto web, pueda ser el cliente final quien haga la gestión de contenidos, dejando al desarrollador para tareas de modificaciones, actualizaciones y mantenimiento.
